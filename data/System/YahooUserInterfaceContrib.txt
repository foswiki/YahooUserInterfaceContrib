---++ Summary of Contents
Contains an un-archived and ready for use copy of YUI 2.8.0 - see http://developer.yahoo.com/yui

If you get the contrib using svn, it will contain all the examples and
documentation that is in the original archive. 

%N% Now also contains a full copy of YUI 3.0.0 - see http://developer.yahoo.com/yui/3/

Note: YUI 3 is able to run on the same page as YUI 2 without any problems.
